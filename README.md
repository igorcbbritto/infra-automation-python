# Infra Automation Python

Projeto em Python para automação de infraestrutura e verificação de disponibilidade de servidores.

## Objetivo

Este projeto foi criado para simular uma rotina de monitoramento simples, verificando se determinados servidores estão online ou offline a partir de uma lista configurada em arquivo JSON.

## Funcionalidades

- Leitura de servidores a partir de arquivo JSON
- Verificação de disponibilidade com ping
- Exibição do status de cada servidor
